# 🌐 ConnectX

A modern Android social media application built using **Kotlin**, **Jetpack Compose**, **Firebase**, and **Material Design 3**.

ConnectX enables users to connect, share moments, discover content, interact with friends, and communicate through a modern and intuitive social networking experience.

---

## ✨ Features

- 🔐 Firebase Authentication
- 👤 User Profiles
- 🏠 Home Feed
- 📝 Create Posts
- 📷 Image Upload
- ❤️ Like & Unlike Posts
- 💬 Comments
- 👥 Follow / Unfollow Users
- 🔍 User Search
- 🔔 Real-time Notifications
- 💌 Real-time Chat
- 🌙 Dark & Light Theme
- 📶 Offline Support
- 🎨 Material Design 3
- ✨ Smooth Animations
